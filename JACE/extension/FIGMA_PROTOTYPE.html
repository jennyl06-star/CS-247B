Interactions & Animations

Smooth animations and transitions that enhance user experience without being intrusive.

Button States
  • Default: Full opacity, solid color
  • Hover: Brighten color, subtle lift (-1px translateY)
  • Active: Move back to baseline, increase shadow
  • Disabled: 35% opacity, no pointer events
Input States
  • Default: Border with opacity 0.08
  • Focus: Green border (#20a565), glow shadow
  • Error: Red border, error message display
  • Valid: Green border with check indicator
Modal Animations
  • Entrance: Fade in overlay (0.3s), scale & slide modal (0.38s cubic-bezier)
  • Scroll: Smooth scrolling with custom scrollbar
  • Exit: Reverse animation on close
  • Timing: 0.2s - 0.38s ease curves
Progress Indicators
  • Dots transition from gray → green → blue
  • Label updates with smooth fade
  • Background color shifts on state change
  • Real-time feedback on interaction

Transition Properties

All transitions: 0.2s ease (buttons, inputs, states) Modal animations: 0.38s cubic-bezier(0.16, 1, 0.3, 1) Color/opacity changes: 0.3s ease Transform/scale: 0.2s ease

Accessibility Standards

WCAG 2.1 AA compliance guidelines and considerations.

Color Contrast Ratios
Primary on Dark
14.2:1 Ratio (AAA)
Secondary on Dark
8.1:1 Ratio (AAA)
Light Blue on Dark
7.9:1 Ratio (AA)
Keyboard Navigation
  • Tab order follows logical flow (top to bottom, left to right)
  • Focus indicators visible with 2px green outline
  • Enter/Space activates buttons and checkboxes
  • Arrow keys navigate through question blocks
  • Escape closes modals
Screen Reader Support
  • Semantic HTML elements (buttons, labels, forms)
  • ARIA labels for icon buttons
  • Form labels explicitly associated with inputs
  • Alert regions for feedback messages
  • Progress indicators described in labels
Text & Readability
  • Minimum font size: 12px for labels, 13px for body
  • Line height: 1.5-1.6 for comfortable reading
  • Line length: Constrained to ~60 characters for readability
  • Letter spacing: 0.04-0.1em for distinction
  • Don't rely on color alone for meaning
