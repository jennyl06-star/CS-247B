# JACE Icons

## Generate Icons

Open `generate-icons.html` in your browser and save the three canvas images as:
- `icon16.png` (16x16 pixels)
- `icon48.png` (48x48 pixels)
- `icon128.png` (128x128 pixels)

Place all three PNG files in this directory.

## Alternative: Use Figma/Design Tool

Create a square icon with:
- **Background**: Green (#20a565)
- **Border Radius**: 22% of size
- **Icon**: Brain emoji 🧠 (60% of icon size, centered)

Export at 16x16, 48x48, and 128x128 resolutions.
